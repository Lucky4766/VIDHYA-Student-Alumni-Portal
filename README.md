## Overview
VIDHYA is a full-stack web application designed to connect students and alumni for mentorship, networking, and career guidance.

## Description
Developed a scalable alumni portal using Java and SQL, enabling users to register, interact, and share opportunities within an academic network. The platform supports structured communication between students and alumni to enhance collaboration and career development.

Implemented backend APIs for user authentication, data management, and role-based access. The application follows a modular architecture, ensuring maintainability and scalability.

The system was containerized using Docker to ensure consistent development and deployment environments. It is designed with cloud deployment in mind, featuring an AWS-ready architecture and Linux-based server configuration.

Optimized backend APIs and database queries, improving application performance and reducing response time by 35%.

## Tech Stack
Java, SQL, HTML, CSS, Docker, Linux, Git

## Key Highlights
- Full-stack web application
- REST API-based backend
- Docker containerization
- AWS-ready deployment architecture
- Performance optimization (35% improvement)

## How to Run
1. Clone the repository  
2. Set up database configuration  
3. Run backend server  
4. Access application via browser
